# About this project 
this tools helps you turn an rce trought php parametr to turn into an interactive shell 


## install 

```bash
$ git clone https://github.com/Ayoub-2/RCE-webshell.git
$ cd RCE-webshell
$ pip install -r requirements
$ python rce.py --help
```

to set up a lab : 
```bash
$ cd docker-example
$ ./build.sh 
```
